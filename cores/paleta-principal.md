# Paleta de Cores Principal

## Cores Primárias
- **Teal Tweed**: #08ffb1 (RGB: 8, 255, 177)
- **Branco**: #FFFFFF
- **Azul-Escuro**: #0b0c13 (RGB: 11, 12, 19)

## Uso
- Teal Tweed: Headlines, CTAs, elementos destacados
- Azul-Escuro: Backgrounds, espaço negativo
- Branco: Texto corpo
